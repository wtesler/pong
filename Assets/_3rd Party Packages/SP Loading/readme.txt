SP Loading is a pack of 3 different loading bars in the same concept, ready for use.

The product features:

- Example Scene
- The complete Photoshop .psd file
- Sliced layers
- Multi-color bars

DEMO - http://unity3d.duloclan.com/sp_loading/

* Please note the JS is just example, made to demonstrate the way loading bars looks like. Anyway feel free to use the script in your project.
* The example script is placed in folder "Script" under the name "GUIProgressBar".
* To disable the animation remove the code inside function Update.

--------------------------------------------------------

GUIProgressBar.js
Component Options:

	Progress Bar GUI Style - the style for the loading bar 
	Font Style Loading - GUI style for the text "Loading...", you can manage its color, size and font...
	Font Style Percentage - GUI style for the Percentage text, you can manage its color, size and font or to hide it
							by setting its normal color alpha to 0...
	
	Margin Left - Moves whole loading bar in this direction
	Margin Top - Moves whole loading bar in this direction
	
	Frame Image - Loading bar frame/background image
	Frame Image Width - The width value of the background image, also of whole loading bar holder
	Frame Image Height - The Height value of the background image, also of whole loading bar holder
	
	Fill Image - The fill image of the loading bar
	Fill Image Width - The width value of the fill image
	Fill Image Height - The height value of the fill image
	Fill Marin Left - Moves only the fill image in this direction
	Fill Marin Top - Moves only the fill image in this direction
	
	Loading Margin Left - Moves the "Loading..." text to left
	Loading Margin Top - Moves the "Loading..." text to top
	
	Perc Margin Left - Moves the percentage text to left
	Perc Margin Top - Moves the percentage text to top
